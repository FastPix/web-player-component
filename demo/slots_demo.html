Named slots demo

Light-DOM children of <fastpix-player> use slot="…" to land in fixed regions (3×3 grid). See SLOTS_DEVELOPER_GUIDE.md. This page is slots only (no quality API). For quality + slots together, use demo/custom_quality_demo.html.

Multiple controls in one region
Use a single element with slot="…" (usually a <div>) and put several buttons inside; style the container with flexbox or grid. That keeps toolbars aligned and makes fullscreen overlays straightforward (one root to pin with position: fixed).

Alternative: several siblings with the same slot name — each light-DOM node with slot="bottom-left" (for example) is assigned to that region in document order. Handy when you generate items in a loop; for a tight row of buttons, a wrapper is usually simpler to lay out.

Live below: bottom-right uses two sibling elements with the same slot (no wrapper). Fullscreen may obscure them unless you pin with script or a wrapper — see Fullscreen behavior in the guide.
